### 데이터셋 구성 확인하기

```r
# 데이터 셋 요약
str(데이터셋 이름)
# 데이터셋 표로 확인
View(데이터셋 이름)
```

---

### 결측값 있는 데이터 제거 (결측값 : 데이터 중 null인 것)

```r
na.omit(데이터셋 이름)
```
